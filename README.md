# TikTok Forces Frontend

React-based frontend for TikTok video processing system.

## Features

- Modern React with TypeScript
- Vite for fast development
- User management UI
- Proxy management
- Video processing interface

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` file:
```
VITE_API_URL=http://localhost:8000
```

## Build

```bash
npm run build
```

## Production

The built files in `dist/` should be served by a web server (Nginx).
